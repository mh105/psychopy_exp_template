# PsychoPy Task Template

This repository is a **minimal template** for a single PsychoPy experiment.

The idea is that when you use this as a template (or clone it), your final
task repo will look something like:

```text
.
├─ resource/                 # all stimuli used in the task
│    ├─ images/              # (optional) image files
│    ├─ sounds/              # (optional) audio files
│    └─ ...                  # any other stimulus files
├─ my_experiment.psyexp      # PsychoPy Builder file
├─ my_experiment.py          # full experiment script (generated by Builder)
├─ .gitignore
└─ README.md
